# Code of Conduct

## Our Standard

Treat other contributors with respect. Critique code and claims directly, not people.

Expected behavior:

- use evidence when making compatibility claims
- assume good intent
- keep review comments concrete
- avoid harassment, insults, and personal attacks

## Enforcement

If someone is disruptive, abusive, or repeatedly ignores project norms, maintainers may remove comments, close threads, or block participation.

## Reporting

Open a private issue or contact the maintainers through GitHub if you need to report behavior that should not be handled in a public thread.
